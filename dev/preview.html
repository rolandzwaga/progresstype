ProgressType

View on GitHub

An OpenType variable font that renders {h:75} as a horizontal progress bar — and {v:75} as a vertical one.

Horizontal bars

Each bar is a fixed-size container; the fill grows inside it. The empty space on the right is the unfilled portion of the track.

CPU{h:23}{h:23}23%
Memory{h:67}{h:67}67%
Disk{h:91}{h:91}91%
Network{h:8}{h:8}8%

In-bar percentage label (ss01)

The font ships a ss01 stylistic set that swaps each prog_h_full_NN for a .label variant — same bar with the percentage punched through it as a knock-out. Enable via font-feature-settings: 'ss01' 1;.

CPU{h:23}{h:23}
Memory{h:67}{h:67}
Disk{h:91}{h:91}
Network{h:8}{h:8}

The label is a fixed pixel size, so at small bar sizes the digits become illegible. Use a CSS container query to drop labels below a usable width: 

.bars-auto-label { container-type: inline-size; }
@container (min-width: 220px) {
  .bars-auto-label .bar { font-feature-settings: 'liga' 1, 'ss01' 1; }
}

Tinting via CSS

The font is monochrome — set color on the bar element (or any ancestor) to tint it. Works with named colours, hex, currentColor, or CSS custom properties tied to your design tokens.

brand {h:65} {h:65}
earth {h:42} {h:42}
design tokens {h:80} {h:80} 
.bar {
  font-family: 'ProgressType', monospace;
  font-feature-settings: 'liga' 1;
}

/* Tint individual bars however you want. */
.bar.brand  { color: #6366f1; }                /* indigo */
.bar.earth  { color: #d97706; }                /* amber-600 */
.bar.tokens { color: var(--brand-primary); }   /* design-token reference */

<p class="bar brand">{h:65}</p>

Corner radius (RADI axis)

Three pre-rendered static instances of the variable font, pinned at RADI 0, RADI 105, and RADI 210. The font also exposes a continuous RADI axis between 0 and 210 if you want to interpolate it yourself.

mid (RADI 105){h:65}{h:65}

Usage

@font-face {
  font-family: 'ProgressType';
  src: url('ProgressType[RADI,wdth,wght].woff2') format('woff2');
  font-weight: 100 900;
  font-stretch: 50% 150%;
}

.bar {
  font-family: 'ProgressType', monospace;
  font-feature-settings: 'liga' 1;
  color: #ef4444;        /* tint with any CSS color */
  /* font-variation-settings: 'RADI' 210; — pill ends */
}

<p class="bar">Disk usage: {h:91}</p>
