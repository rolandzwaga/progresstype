Datatype Progress

An OpenType variable font that renders {h:25,10,15} as a stacked horizontal progress bar — and {v:75} as a vertical one.

Single-segment

CPU{h:23}23%
Memory{h:67}67%
Disk{h:91}91%
Network{h:8}8%

Stacked multi-segment

Up to 8 segments; each segment's color is determined by its position in the list.

2 segments{h:30,40}70%
3 segments{h:25,10,15}50%
4 segments{h:20,15,10,25}70%
8 segments{h:10,10,10,10,10,10,10,10}80%

Same data, RAG palette

Status{h:45,30,25}on track / at risk / blocked

Usage

@font-face {
  font-family: 'DatatypeProgress';
  src: url('DatatypeProgress[wdth,wght].woff2') format('woff2');
  font-weight: 100 900;
  font-stretch: 50% 150%;
}

@font-palette-values --rag {
  font-family: 'DatatypeProgress';
  base-palette: 1;       /* 0=sequential, 1=rag, 2=categorical, 3=ocean, 4=mono */
}

.bar {
  font-family: 'DatatypeProgress', monospace;
  font-feature-settings: 'liga' 1, 'calt' 1;
  font-palette: --rag;
}

<p class="bar">Status: {h:45,30,25}</p>
